# autokey
 跨平台键盘~~鼠标~~宏工具（脚本）
    
    python 版本 3.9.2
    依赖库 pynput, tkinter
    测试环境 MacOS

### UI工具模式

```python
    sudo python3 autokeyUI.py
```
### 脚本模式

```python
    sudo python3 autokey.py
```

### 功能：
* 录制并编辑键盘、鼠标动作记录
* 自动执行动作记录
* 支持快捷键操作
